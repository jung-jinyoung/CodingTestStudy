## 코드
정확성 100점! 효율성 0점..

```python
def solution(k, room_number):
    answer = []
    
    for num in room_number:
        i = num
        while i <= k:
            if i not in answer:
                answer.append(i)
                break
            i += 1
    
    return answer
```
