# [이모티콘 할인행사](https://school.programmers.co.kr/learn/courses/30/lessons/150368)
## 푼 방법
모든 이모티콘 할인 행사 할 수 있는 경우의 수들을 구한 후 모두 완전 탐색!

## 코드
```python
from itertools import product

def solution(users, emoticons):
    def user_cost(want_discount, min_cost, case): 
        total_cost = 0
        for i, discount in enumerate(case):
            if discount >= want_discount:
                total_cost += emoticons[i] * (100 - discount) / 100
        return total_cost

    N = len(users)  # 유저 수
    M = len(emoticons)  # 이모티콘 수
    
    # 이모티콘 할인율 경우의 수 생성
    cases = product([10, 20, 30, 40], repeat=M)
    
    result = [0, 0]  # [유저 수, 총 매출]
    
    for case in cases:  # case = (10, 10, 40) 같은 형태
        temp = [0, 0]  # [유저 수, 총 매출]
        for want_discount, min_cost in users:
            cost = user_cost(want_discount, min_cost, case)
            if cost >= min_cost:
                temp[0] += 1  # 유저가 원하는 비용 이상이면 만족하는 유저 수 증가
            else:
                temp[1] += cost  # 총 매출 업데이트
        
        if temp[0] > result[0]:
            result = [temp[0], temp[1]]
        elif temp[0] == result[0] and temp[1] > result[1]:
            result[1] = temp[1]
    
    return result
```
